# SalisEngine.AI Design System v2.0
> **Theme Name**: Deep Space & Corporate Steel  
> **Version**: 2.0.0 (Logo-Aligned)  
> **Framework**: Next.js 14 + Tailwind (Conceptual) / CSS Modules

## 🌌 Design Philosophy
The **SalisEngine** aesthetic is built to feel like a high-performance "Command Center" for autonomous AI. It combines the sleek, calculated precision of **corporate steel** with the futuristic, infinite depth of **deep space**.

The palette is strictly derived from the official logo:
*   **Sky Blue (`#0ea5e9`)**: Represents intelligence, clarity, and the core engine.
*   **Vibrant Orange (`#f97316`)**: Represents action, execution, and critical data points.
*   **Deep Void (`#050510`)**: A backdrop that reduces eye strain and provides contrast for glowing elements.

## ✨ Effects & Glassmorphism

The design relies heavily on "Physical Glass" — glass that feels substantial, not just blurry.

### Glass Variables
```css
:root {
  --glass-border: rgba(255, 255, 255, 0.08);    /* Subtle white edge */
  --glass-surface: rgba(255, 255, 255, 0.03);   /* Base transparency */
  --glass-highlight: rgba(255, 255, 255, 0.08); /* Top-down lighting */
  --glass-heavy: rgba(15, 23, 42, 0.85);        /* almost opaque */
}
```

### Motion & Glows
*   **Hover State**: Elements should "lift" (`y: -5px`) and "glow" upon interaction.
*   **Glow Variable**: `--glow-primary: 0 0 25px rgba(14, 165, 233, 0.3);`

## 🧩 Component Library

### 1. `GlassCard`
*   **Path**: `app/components/ui/GlassCard.js`
*   **Purpose**: The standard container for all content.
*   **Features**:
    *   Backdrop blur (`16px`)
    *   Subtle top-to-bottom transparency gradient.
    *   Hover lift physics (optional).

### 2. `GlowButton`
*   **Path**: `app/components/ui/GlowButton.js`
*   **Purpose**: Primary call-to-action interactions.
*   **Variants**: 
    *   `primary`: Uses the Blue Gradient (`--gradient-main`).
    *   `danger`: Uses Red/Error colors.
    *   `ghost`: Transparent background with border.

### 3. `Console`
*   **Path**: `app/components/Console.js`
*   **Purpose**: A dedicated feedback terminal.
*   **Styling**: JetBrains Mono font, dark slate background (`#0d1117`), macOS-style window controls.

### 4. `Sidebar`
*   **Path**: `app/components/Sidebar.js`
*   **Purpose**: Main navigation.
*   **Features**:
    *   Displays the Salis.AI Logo.
    *   Active tabs show a "Neon Strip" on the left and a radial glow background.

## 🛠️ Usage Guide

### Creating a New Page/View
1.  Wrap your content in `<main className="flex-1 ...">`.
2.  Use `GlassCard` for grouping related content.
3.  Use `GlowButton` for actions.

### Adding a New Color
Always add new colors to `globals.css` under the `:root` scope to maintain consistency. Do not hardcode hex values in components.

```css
:root {
  --new-color: #......;
}
```
