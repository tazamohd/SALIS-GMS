"Forcefully apply the SalisAuto UI/UX Design System for a futuristic, high-performance 'Command Center' experience. The design should reflect the sleek and modern corporate steel aesthetic combined with the infinite depth of deep space, creating a dark and focused environment that reduces eye strain while highlighting interactive elements with glowing accents.

Key Design Specifications:

Background & Surfaces:

Primary Background: Use Dark Obsidian with hex #0B0F0D or #0E1411 for the canvas. These near-black tones provide a futuristic and focused workspace.

Surface & Card Elements: Use Charcoal Panels (#141A17 to #1C2420) for cards and side panels to create subtle depth with no borders.

Brand & Action Colors:

Primary Brand Accent: Apply Saudi Green (#22C55E) for all primary actions, interactive elements, and active states. Use sparingly as a highlight for maximum impact.

Secondary Accent: Use Electric Yellow/Gold (#EAB308) for performance badges, warnings, and important alerts.

Semantic Layer:

Success: Green (#22C55E)

Warning: Amber (#F59E0B)

Error: Soft Red (#EF4444)

Info: Cyan/Teal (#06B6D4)

Text & Typography:

Primary Text: Soft white (#E5E7EB) for high contrast on dark backgrounds.

Secondary Text: Muted gray (#9CA3AF) for secondary and meta information.

Font: Use Inter for English text and Cairo or Tajawal for Arabic/RTL support.

Data Visualization:

Primary Line in Charts: Green (#22C55E) for key metrics and trends.

Secondary Line: Gold (#EAB308) for secondary metrics.

Chart Fill Gradient: Green to transparent black for depth and glow effects.

Grid Lines: Ultra-low opacity gray (#FFFFFF10) for a subtle background.

Glow & Shadow Effects:

Green Glow: Apply rgba(34,197,94,0.25) for active states, live indicators, and interaction feedback.

Gold Glow: Apply rgba(234,179,8,0.25) for performance-related actions and alerts.

Shadow Effects: Use a green-tinted black shadow (0 0 24px rgba(34,197,94,0.25)) for depth.

Layout & Spacing:

RTL Support: Implement RTL layout structures for Arabic viewers, flipping icons and ensuring that all directional elements align correctly.

Spacing: Maintain a consistent 8px base unit for all margins and paddings. Utilize utility classes for standard layouts and responsive designs.

Interactive States & Transitions:

Hover Effects: Use subtle hover lifts and animations to create a tactile feel for interactive elements like cards and buttons.

Focus & Active States: Highlight active and focused elements with a soft green or gold glow, ensuring accessibility with proper focus-visible indicators.

Button Transitions: Smooth transitions between states, with slight scaling and background color shifts to emphasize user interaction.

Accessibility & Compliance:

Ensure all elements adhere to WCAG 2.1 AA accessibility standards, providing a visually accessible interface for users with disabilities.

Optimize touch targets to meet a minimum size of 44px by 44px for all interactive components.

Apply the system using the SalisAuto Design Tokens, ensuring consistency across all pages and components. The overall look and feel should reflect the high-tech, government-grade interface with a sharp focus on control, authority, and performance, suitable for Vision 2030 style smart city control rooms and enterprise-level operations."
