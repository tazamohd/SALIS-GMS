TASK: Update SalisAuto UI Theme

INSTRUCTIONS:
- Change current UI theme to "SalisAuto Vision Theme" with these core color definitions.
- Apply to all system interfaces (admin dashboards, client-facing panels, analytics).

COLOR PALETTE TO BE APPLIED:

1️⃣ Primary Base (Background)
  - Use **Dark Obsidian / Near-Black** for app backgrounds (HEX: #0B0F0D → #0E1411).
  - Role: Main canvas and background.
  - Rationale: Reduces eye strain, maximizes contrast, and enhances neon accents.

2️⃣ Primary Brand Accent (Saudi Green)
  - Use **Emerald Neon Green** (HEX: #1ED760 → #22C55E) for:
    - Active states (e.g., buttons, interactive elements)
    - Success indicators (e.g., charts, metrics)
    - Online status markers
  - **Usage Rule**: This is a *highlight accent*—use sparingly in key UI elements.

3️⃣ Secondary Accent (Gold for Performance)
  - Use **Electric Yellow / Gold** (HEX: #FACC15 → #EAB308) for:
    - Efficiency badges
    - Warnings
    - Highlighting key metrics (e.g., 99.8% efficiency)
  - **Warning**: Do not overuse this color—it should stay as an accent to avoid clutter.

4️⃣ System Neutral Cards
  - Use **Charcoal Panels** (HEX: #141A17 → #1C2420) for:
    - Card backgrounds
    - Container panels
    - Side navigation areas
  - Design rule: Ensure the cards are slightly lighter than the background for depth, with no borders.

5️⃣ Text System (Typography)
  - **Primary Text**: Use **Soft White** (HEX: #E5E7EB)
  - **Secondary Text**: Use **Muted Gray** (HEX: #9CA3AF)
  - **Disabled / Meta Text**: Use **Cool Gray** (HEX: #6B7280)
  - **Note**: Never use pure white; ensure readability.

6️⃣ Status Colors (Semantic Layer)
  - **Success**: **Emerald Green** (HEX: #22C55E)
  - **Warning**: **Amber** (HEX: #F59O0B)
  - **Error**: **Soft Red** (HEX: #EF4444)
  - **Info**: **Cyan/Teal** (HEX: #06B6D4)

7️⃣ Data Visualization (Charts & Graphs)
  - **Primary Line Color**: **Green** (HEX: #22C55E)
  - **Secondary Line Color**: **Gold** (HEX: #EAB308)
  - **Fill Gradient**: Green to transparent black (to ensure a glowing effect)
  - **Grid Lines**: Use ultra-low opacity gray (HEX: #FFFFFF10)

8️⃣ Glow & Depth Effects
  - **Glow Effects**:
    - **Green glow**: Use to denote active/alive elements (HEX: rgba(34,197,94,0.25))
    - **Yellow glow**: Use for performance/alert indicators (HEX: rgba(234,179,8,0.25))
  - **Shadow Color**: Use **green-tinted black** for shadows (HEX: rgba(0,0,0,0.45)).

DESIGN PHILOSOPHY:
- **Dark ≠ boring**: The theme is focused and modern, with a clean, high-tech feel.
- **Green ≠ decoration**: It signals authority and active states.
- **Yellow ≠ color**: It catches attention without overwhelming.
- **Contrast is intentional**: It’s bold and functional for high-authority interfaces.

TASK EXECUTION:
- Begin applying the above settings.
- Ensure proper testing across all system pages (admin, client, dashboard).
- Confirm final implementation meets the aesthetic and functional needs.
