# SALIS AUTO Screenshots — Appointments

This folder contains screenshots for the **Appointments** module of the SALIS AUTO platform.

## Naming Convention
`[module]-[screen-name]-[view].png`

## Capture Standards
- **Resolution**: 1920×1080
- **Format**: PNG
- **Theme**: Dark mode
- **Sample Data**: Must be visible
