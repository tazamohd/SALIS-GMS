# Accounting Module Screenshots

This folder contains screenshots for the 10 accounting modules in SALIS AUTO.

## Required Screenshots

### 1. General Ledger (دفتر الأستاذ العام)
- `/general-ledger` - Main view with chart of accounts
- Features: Account hierarchy, balances, transaction history

### 2. Journal Entries (القيود اليومية)
- `/journal-entries` - Journal entry list and creation form
- Features: Debit/credit entries, templates, approval status

### 3. Trial Balance (ميزان المراجعة)
- `/trial-balance` - Balance report view
- Features: Date range picker, debit/credit columns, totals

### 4. Income Statement (قائمة الدخل)
- `/income-statement` - P&L report view
- Features: Revenue, expenses, net income, period comparison

### 5. Balance Sheet (الميزانية العمومية)
- `/balance-sheet` - Financial position report
- Features: Assets, liabilities, equity sections

### 6. Cash Flow Statement (قائمة التدفقات النقدية)
- `/cash-flow-statement` - Cash flow report view
- Features: Operating, investing, financing activities

### 7. Accounts Receivable (حسابات المدينين)
- `/accounts-receivable` - AR aging and customer balances
- Features: Aging buckets, customer list, overdue alerts

### 8. Accounts Payable (حسابات الدائنين)
- `/accounts-payable` - AP aging and vendor balances
- Features: Vendor invoices, payment scheduling, aging

### 9. Cost Centers (مراكز التكلفة)
- `/cost-centers` - Cost center management
- Features: Department costs, allocation, profitability

### 10. Budget Management (الميزانية التقديرية)
- `/budget-management` - Budget planning and tracking
- Features: Budget vs actual, variance analysis, forecasts

## Screenshot Guidelines

1. **Resolution**: 1920x1080 or higher
2. **Format**: PNG with transparent background where applicable
3. **Naming Convention**: `module-name-view.png` (e.g., `general-ledger-main.png`)
4. **Content**: Use sample data, avoid personal information
5. **Theme**: Dark theme preferred (as per SALIS AUTO design system)
6. **Language**: Include both English and Arabic screenshots where applicable

## How to Capture

1. Navigate to the accounting module
2. Ensure sample data is visible
3. Use browser screenshot tool or Replit's capture feature
4. Save to this folder with appropriate naming
