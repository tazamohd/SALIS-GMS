# Screenshots Directory

This folder contains screenshots of the SALIS AUTO platform.

## Organization

Screenshots are organized by module/feature:

- `dashboard/` - Dashboard and overview screens
- `accounting/` - Accounting module screenshots
- `inventory/` - Inventory and parts management
- `customers/` - Customer management screens
- `vehicles/` - Vehicle management screens
- `invoicing/` - Invoicing and billing
- `reports/` - Analytics and reporting

## Naming Convention

Use descriptive names with the following format:
`[module]-[screen]-[description].png`

Examples:
- `accounting-general-ledger-overview.png`
- `dashboard-kpi-cards.png`
- `inventory-parts-list.png`

## Adding New Screenshots

1. Navigate to the relevant page in the application
2. Take a screenshot (recommended size: 1920x1080)
3. Save in the appropriate subfolder
4. Use PNG format for clarity
5. Update this README if adding new categories
