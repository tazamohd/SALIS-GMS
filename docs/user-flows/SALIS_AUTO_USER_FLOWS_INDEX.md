## Document Conventions

- **User Flow**: End-to-end journey a user takes to complete a task
- **Scenario**: Specific use case with preconditions and outcomes
- **System Flow**: Backend process involving multiple systems/services
- **Integration Flow**: Data exchange between SALIS AUTO and external systems
