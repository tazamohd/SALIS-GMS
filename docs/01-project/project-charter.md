## Project Purpose

SALIS AUTO was initiated to address the critical lack of a comprehensive, Arabic-native, ZATCA-compliant automotive ERP platform in the Saudi Arabian and MENA region. The project aim is to build the most feature-complete automotive management platform in the region, enabling garages of all sizes to operate at enterprise scale.

---

## Project Objectives

1. Build a unified platform covering 100% of garage operations from customer intake to financial reporting
2. Achieve full Saudi regulatory compliance (ZATCA, VAT, Zakat)
3. Support Arabic language with complete RTL layout
4. Integrate AI capabilities for predictive operations
5. Enable franchise and multi-location management
6. Deliver dedicated portals for all 20+ user roles
7. Launch dedicated mobile applications for technicians and customers

---

## Project Scope

### In Scope
- All 14 development phases as documented in the Roadmap
- 235+ screens and 156+ modules
- 320+ database tables
- 1,000+ API endpoints
- Full Arabic translation (2,000+ keys)
- Saudi compliance stack (ZATCA, VAT, Zakat)
- Three mobile applications (Technician, Customer, Manager)
- AI integration (OpenAI GPT-4o)
- Emerging technologies (AR, VR, Blockchain, IoT, Digital Twin)

### Out of Scope
- Native iOS/Android apps (covered by PWA and mobile web)
- Point-of-sale hardware integration
- Vehicle telematics hardware (integration only, not hardware)
- Custom OEM software development

## Constraints

1. **Regulatory:** Must comply with ZATCA Phase 1 and Phase 2 requirements
2. **Language:** Full Arabic support required from day one
3. **Technology:** Must run on modern web browsers without native app requirement
4. **Performance:** Sub-2 second response time for all API calls
5. **Security:** Role-based access control for all 20 user roles
