## Screen 137 — Staff Scheduling (`/staff-scheduling`)

Work schedule management for all staff.

### Scheduling Features
- **Weekly calendar view** per branch

## Screen 138 — Staff Performance Review (`/staff-performance-review`)

### Description
Structured performance evaluation system.

### Review Process
```
Review Period End (Quarterly/Annual)
→ HR creates review for each employee
→ Manager fills in review form (competencies, goals, ratings)
→ Employee reviews and signs
→ Salary adjustment recommendation
→ HR approves and files
```

---

## Screen 139 — Timesheet Management (`/timesheet-management`)

### Description
Review and approval of staff timesheets.

### Timesheet Approval Workflow
```
Staff submits timesheet (end of week)
→ Manager reviews hours
→ Check against time clock data
→ Approve or request correction
→ Approved timesheet → Payroll calculation
```

## Screen 140 — Timeclock & Payroll (`/timeclock-payroll`)

### Description
Integrated time-tracking and payroll processing hub.

### Features
- Real-time time clock data for all staff
- Overtime calculation (1.5x after 8 hours, 2x on holidays)
- Absence and late arrival tracking
- Payroll preview before processing

---

## Screen 141 — Payroll Management (`/payroll-management`)

### Description
Payroll run processing and salary management.

### Payroll Run Process
1. Select pay period
2. Review time and attendance data
3. Check leave deductions
4. Preview salary slips for each employee
5. Approve and process payroll
6. Generate bank transfer file (SADAD format for Saudi banks)
7. Send salary slips to employees

## Screen 143 — Training LMS (`/training-lms`)

### Description
Learning Management System for staff training and certification.

### Training Content
- **Onboarding courses** — Required for all new hires
- **Technical courses** — Vehicle-specific training
- **Safety training** — OSHA-equivalent courses (required annually)
- **Soft skills** — Customer service, communication
- **Compliance** — VAT, ZATCA procedures

### LMS Features
- Video lectures with progress tracking
- Quizzes and assessments
- Completion certificates (downloadable)
- Manager dashboard (completion rates)
- Course creation (upload your own content)

## Screen 144 — Wearable Integration (`/wearable-integration`)

### Description
Integration with wearable devices for technician safety monitoring.

### Supported Devices
- Smart safety glasses (AR overlay)
- Smart gloves (touch-screen compatible)
- Safety vests (location tracking in workshop)
- Heart rate monitors (fatigue detection)

### Safety Applications
- **Fall detection** — Alert if technician falls
- **Location tracking** — Know which bay each technician is in
- **Fatigue monitoring** — Alert manager if heart rate unusually high
- **Emergency alert** — One-touch emergency button
