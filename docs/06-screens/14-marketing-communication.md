## Screen 173 — Marketing Automation (`/marketing-automation`)

### Description
Automated marketing workflow builder.

### Workflow Examples
```
Trigger: Customer hasn't visited in 90 days
→ Action: Send "We miss you" SMS with 15% discount
→ Condition: If no response in 7 days
→ Action: Follow-up email with different offer
```

```
Trigger: Customer's vehicle hits mileage milestone (50,000 km)
→ Action: Send major service reminder
→ Action: Offer appointment booking link
```

## Screen 178 — Call Center (`/call-center`)

### Description
Integrated call center management for customer service operations.

### Features
- **Call logging** — Record all inbound/outbound calls
- **Customer popup** — Auto-load customer profile when they call
- **Call scripts** — Guided scripts for common inquiries
- **Callback scheduling** — Schedule follow-up calls
- **Call recordings** — Store recordings for training
- **Performance metrics** — Average handle time, call volume

### Call Center KPIs
- Calls handled per agent per hour
- Average handle time
- First call resolution rate
- Customer satisfaction post-call
- Missed call rate
