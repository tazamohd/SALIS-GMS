# Screen Documentation — Section 11: Reports & Analytics

**Screens:** 127–134  
**Section:** Analytics & Business Intelligence  
**Navigation Group:** Analytics & Business Intelligence  

---

## Overview

The Analytics section transforms raw garage data into actionable business intelligence. From simple reports to advanced heat maps and KPI dashboards, these tools help management make data-driven decisions.

---

## Screen 127 — Reports (`/reports`)

### Description
Central reports hub with quick access to all standard reports.

### Report Categories
| Category | Reports Available |
|----------|-----------------|
| Financial | Revenue, Profit, Cash Flow, VAT Summary |
| Operations | Job Volume, Completion Rate, Bay Utilization |
| Customers | New vs. Returning, Satisfaction, Loyalty |
| Inventory | Stock Turns, Reorder Report, Parts Usage |
| Staff | Technician Productivity, Hours Worked |
| Compliance | VAT Return, ZATCA Submission Status |

### Report Generation
1. Select report type
2. Set date range
3. Filter by branch (all or specific)
4. Generate → View in browser
5. Export as PDF, Excel, or CSV

---

## Screen 128 — Custom Reports (`/custom-reports`)

### Description
Report builder for creating custom analytics views.

### Report Builder Features
- **Drag-and-drop columns** — Add any field from any module
- **Filter builder** — Complex multi-condition filters
- **Grouping** — Group by date, category, technician, etc.
- **Aggregation** — Sum, average, count, min, max
- **Visualization** — Bar, line, pie chart options
- **Save report** — Reuse named reports
- **Schedule** — Auto-send reports via email on a schedule

## Screen 130 — Business Intelligence Dashboard (`/business-intelligence-dashboard`)

### Description
Executive BI dashboard with the most important cross-functional metrics.

### Dashboard Panels
1. **Revenue Overview** — Total, per service type, vs. target
2. **Customer Metrics** — NPS, satisfaction, churn rate
3. **Operational Efficiency** — Bay utilization, tech efficiency
4. **Financial Health** — Gross margin, cash position, AR aging
5. **Growth Indicators** — New customers, repeat rate, referrals

---

## Screen 131 — Business Heat Maps (`/business-heatmaps`)

### Description
Geographic and temporal heat map visualizations for pattern discovery.

### Heat Map Types
- **Hourly Demand** — Which hours are busiest (helps with staffing)
- **Day of Week** — Which days have most vehicles
- **Service Type by Season** — AC service spikes in summer
- **Customer Location Map** — Geographic customer distribution
- **Revenue by Service Type** — Color-coded service profitability

---

## Screen 132 — Profit Analysis (`/profit-analysis`)

### Description
Detailed profitability analysis by service type, technician, and time period.

### Profit Metrics
- **Gross Profit** — Revenue minus cost of parts and labor
- **Net Profit** — After overhead allocation
- **Margin %** — Per service type, per job, per technician
- **Labor Efficiency** — Billed hours vs. worked hours
- **Parts Margin** — Markup percentage on parts

## Screen 134 — Productivity Tracker (`/productivity-tracker`)

### Description
Technician and team productivity monitoring with efficiency scoring.

### Metrics Per Technician
- Jobs completed this week/month
- Total hours billed
- Efficiency ratio (billed/available hours)
- Quality score (rework rate)
- Parts accuracy (correct parts on first request)

### Team Metrics
- Combined workshop throughput
- Average job cycle time
- Queue length trend
