## Project Overview

SALIS-GMS ("SALIS AUTO") is an automotive ERP / **Garage Management System** for single-location and multi-tenant franchise garage networks. It spans customers/vehicles, appointments, job cards, technician assignment & tracking, inventory/parts, invoicing/payments, reporting, and customer/technician portals — with Saudi-market compliance (ZATCA e-invoicing, VAT, Zakat, Hijri) and a 7-language UI.

## Getting Started

### Prerequisites

- Node.js 20+, PostgreSQL 16 (local via `docker-compose` or a Neon `DATABASE_URL`)
- `.env` with `DATABASE_URL` and `SESSION_SECRET` (validated at startup — server exits if missing)

### Setup

```bash
npm install
cp .env.example .env      # set DATABASE_URL and SESSION_SECRET
npm run db:push           # apply Drizzle schema
npm run db:seed           # seed sample data + compliance/config tables
```

### Run Locally

```bash
npm run dev               # Express + Vite HMR on http://localhost:5000
```

### Run Tests

```bash
npm run test              # Vitest (unit/integration)
npx playwright test       # Playwright E2E (e2e/) — needs the app on :5000
```
